# Contributing to xMoney Node.js SDK

Thank you for your interest in contributing to the xMoney Node.js SDK! We welcome contributions from the community.

## Getting Started

1. **Fork the repository** on GitHub.
2. **Clone your fork** locally: `git clone https://github.com/YOUR_USERNAME/xmoney-node.git`
3. **Install dependencies**: `npm install`
4. **Create a branch** for your feature or bug fix: `git checkout -b my-new-feature`

## Development

- **Linting**: Run `npm run lint` to ensure your code matches our coding standards.
- **Testing**: Run `npm test` to run the test suite. Please add new tests for any new functionality.
- **Building**: Run `npm run build` to compile the TypeScript code.

## Pull Requests

1. Push your branch to GitHub.
2. Submit a Pull Request to the `main` branch of the `xmoney-payments/xmoney-node` repository.
3. Provide a clear description of your changes.
4. Ensure all tests pass and the linter is happy.
