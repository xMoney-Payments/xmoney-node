# Changelog

All notable changes to this project will be documented in this file.

## [1.1.0] - 2025-12-06

- Comprehensive CLI support for `Cards`, `Orders`, `Customers`, and `Transactions` resources.
- Support for missing fields in `Orders` (e.g., `intervalType`, `cardNumber`) and `Transactions` resources to match API specification.
- Unit tests for CLI commands.

## [1.0.0] - 2025-12-01

- Initial release of xMoney Node.js SDK.
- Support for basic `Cards`, `Orders`, `Customers`, `Transactions` operations.
- CLI interface for interacting with the API.
